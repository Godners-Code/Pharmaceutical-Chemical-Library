# 前列地尔溶剂配制

- [返回首页](../README.md)
- 另请参见：[前列地尔](../../Hormonal_Balance_Compendium/Vitality_Source_Notes/PGE1.md)

## 先按你给定条件做严格换算

- 前列地尔总量：5 mg
- 目标浓度：20 μg/mL = 0.02 mg/mL
- V = \frac{5\ \text{mg}}{0.02\ \text{mg/mL}} = 250\ \text{mL}
- 配方最终是
  - 总体积：250 mL
  - 前列地尔：5 mg
  - 浓度：20 μg/mL

## 按 PEG400 : 甘油配比拆分

- 安全注射方案
  - **PEG400 : 甘油 = 4 : 1**
  - PEG400 = 250 × 80% = 200 mL
  - 甘油 = 250 × 20% = 50 mL
- 平衡方案方案
  - **PEG400 : 甘油 = 3 : 1**
  - PEG400 = 250 × 75% = 187.5 mL
  - 甘油 = 250 × 25% = 62.5 mL
- 黏附尿道方案
  - **PEG400 : 甘油 = 1 : 3**
  - PEG400 = 62.5 mL
  - 甘油 = 187.5 mL

## PEG/甘油体系的真实问题

- 高黏度 → 注射扩散不均
- 高渗透压 → 局部细胞应激
- 长期稳定性仍未知（前列地尔仍会氧化/异构化）
- 单次注射必须反推，例如
  - 0.5 mL 注射，20 μg/mL
  - 则每次只有 $0.5 \times 20 = 10\ \mu g$
  - 这在药理上属于非常低剂量区间

## 总结

- 如果严格按你设定浓度
  - 总体积必须：250 mL
  - PEG400：187.5–200 mL（推荐区间）
  - 甘油：50–62.5 mL（推荐区间）
